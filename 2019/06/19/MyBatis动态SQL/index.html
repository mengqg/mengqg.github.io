1.1.1 动态SQL简介

  • 动态SQL是MyBatis框架中强大特性之一。在一些组合查询页面，需要根据用户输入的查询条件生成不同的查询SQL，这在JDBC或其他相似框架中需要在代码中拼写SQL，经常容易出错，但是在MyBatis中可以解决这种问题。
  • 使用动态SQL元素与JSTL相似，它允许我们在XML中构建不同的SQL语句。常用的元素如下：
    • 判断元素：if,choose
    • 关键字元素：where,set,trim
    • 循环元素：foreach
